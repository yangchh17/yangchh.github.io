# A Living GIS Portfolio (Under Development) — Chenghao Yang 

A personal portfolio website showcasing geospatial projects, spatial data workflows, and GIS engineering work.

Built with a lightweight static HTML architecture and deployed through **GitHub Pages**.

🔗 Live site: [https://chhyang17.github.io](https://chhyang17.github.io/yangchh.github.io)

---

# Overview

This repository contains the source code for a GIS-focused portfolio site presenting selected professional projects including:

- Infrastructure asset monitoring systems
- Airport obstacle limitation surface (OLS) modelling
- LiDAR terrain analysis and hydrological modelling
- Quarry stockpile volume estimation
- Geospatial ETL and geodatabase automation
- Regional cartographic atlas production

The goal of this portfolio is to demonstrate **end-to-end spatial problem solving**, from raw geodata processing to decision-support outputs.

---

# Featured Projects

## Infrastructure Asset Monitoring
Field-to-office GIS workflow built using Survey123 and ArcGIS Dashboards to centralize infrastructure condition monitoring across a remote community.

Key components:
- Survey123 field data collection
- Dashboard reporting
- Asset condition tracking
- GIS database management

---

## Airport OLS & Hazard Tree Detection
3D modelling of airport obstacle limitation surfaces (ICAO Annex 14) combined with LiDAR-based individual tree detection to identify aviation hazards.

Key components:
- LiDAR processing
- OLS surface modelling
- hazard identification
- timber volume estimation

---

## Bioregional Atlas Map Series
Multi-theme printed atlas supporting Indigenous consultation and regional planning.

Key components:
- ArcGIS Pro map series
- cartographic design
- environmental datasets integration

---

## Terrain Modeling & Hydrological Analysis
Reproducible LiDAR processing workflow built in **R** using the `lidR` package.

Outputs include:
- high-resolution DTMs
- watershed delineation
- stream network extraction
- Strahler stream order classification

---

## Stockpile Volume Estimation
Drone and LiDAR based volumetric analysis used for quarry inventory estimation.

Key components:
- DSM / DTM differencing
- cut-fill analysis
- drone photogrammetry integration

---

## Geospatial ETL & Geodatabase Automation
Design of automated pipelines that synchronize datasets across multiple engineering and environmental projects.

Key tools:
- Python
- FME
- ArcGIS ModelBuilder
- enterprise geodatabase design
