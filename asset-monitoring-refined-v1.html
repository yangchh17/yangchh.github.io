Project 01

Infrastructure Asset
Monitoring System

Centralized 100+ infrastructure assets for a remote First Nations community into a real-time monitoring platform — replacing fragmented spreadsheets with ArcGIS Pro geodatabase design, Survey123 field capture, and Python-automated reporting. Cut federal report preparation by ~75% and established a replicable framework for community-driven asset management across Indigenous communities.

Domain Indigenous Infrastructure
Scale 100+ Assets
Role GIS Specialist
Status Deployed
Spatial Context // Asset Coverage
00
Context
  • → 100+ infrastructure assets (buildings, water/wastewater, boardwalk roads, utilities, fuel storage, electrical) across a remote First Nations community required regular federal condition assessments
  • → Visiting engineers conducted infrequent inspections — no system existed to track asset conditions or maintenance status between visits
  • → Data lived in scattered engineering reports, spreadsheets, and institutional memory with zero spatial integration
01
Problem
  • → Zero single source of truth — decision-makers couldn't access current asset status without manually reconciling fragmented records across multiple formats
  • → Federal reporting cycles required weeks of manual data compilation before each submission — pulling from disparate spreadsheets, PDFs, and field notes
  • → No spatial context made it impossible to identify patterns across asset categories or communicate status to non-technical community stakeholders
02
Approach
  • → Deployed ArcGIS Survey123 forms aligned to the ACRS framework — standardizing field capture of condition ratings, photos, and maintenance notes across 7 asset categories
  • → Designed a geodatabase schema in ArcGIS Pro with domains and subtypes enforcing data integrity — serving as the single source of truth for all downstream reporting and Python-driven data exports
  • → Built ArcGIS Dashboards with custom Arcade expressions — real-time condition overviews, asset-level drill-downs, and visual summaries accessible to field staff, program managers, and community leadership without GIS expertise
