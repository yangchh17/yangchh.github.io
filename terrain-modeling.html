Project 04a

LiDAR-Based
Terrain Modeling

High-resolution terrain surface generation from airborne LiDAR — producing DTM, DSM, and derived surfaces for slope, aspect, hillshade, and curvature analysis across complex landscapes.

MethodLiDAR Processing

00
Context

Terrain analysis is a foundational component of natural resource management, hazard assessment, and environmental planning. High-density airborne LiDAR provides the raw point cloud data needed to generate centimetre-accurate bare-earth models — but realizing that accuracy requires careful classification, filtering, and surface interpolation across varying terrain complexity.

01
Problem

Standard provincial DEM products (25m or coarser) are insufficient for site-level engineering, hydrology, or vegetation structure analysis. This project required generating a complete suite of analysis-ready terrain products from raw LiDAR point cloud data over a multi-tile survey area in central BC — with full traceability from raw point cloud to derived surfaces.

02
Approach
Phase A — Point Cloud Classification

Ground/non-ground classification using LAStools (lasground) with adaptive TIN parameters tuned for mixed forest and steep terrain. Quality checks performed using point cloud density analysis and cross-validation against field GPS check points.

Phase B — Surface Generation

DTM interpolated via TIN (triangulated irregular network) at 1m resolution. DSM derived from first-return maximum height within each cell. nDSM (normalized DSM) computed as DSM − DTM, representing canopy height above ground.

Phase C — Derived Products

Slope, aspect, hillshade (multi-directional), curvature (profile and plan), and topographic wetness index (TWI) computed from the DTM. Products exported as GeoTIFF for downstream analysis and delivered as a project geodatabase.
