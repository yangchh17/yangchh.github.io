Infrastructure Asset Monitoring System

Overview

Designed a field-to-office GIS workflow to monitor infrastructure assets in a remote Indigenous community. The system centralized asset condition data and enabled near real-time monitoring.

Context

Infrastructure assets in the community included buildings, water systems, wastewater systems, boardwalk roads, utilities, fuel storage, and electrical infrastructure. Asset assessments were conducted by visiting engineers as part of a federal reporting program.

Challenge

Before the system was implemented, asset data was scattered across reports and spreadsheets. There was no centralized GIS database or consistent workflow to track asset conditions.

Approach

A GIS workflow was developed using Survey123 for standardized field data collection. A centralized asset database schema was designed based on the ACRS reporting framework, and ArcGIS Dashboards were created to visualize infrastructure condition data.

Outcome

The system established a centralized asset monitoring platform covering over 100 infrastructure assets. The workflow improved data consistency and enabled faster access to asset condition information.
